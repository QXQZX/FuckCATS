## 银行家算法

建议使用 [Banker.cpp](Banker.cpp)

#### 使用步骤

1. 初始化数据
2. 显示资源分配情况(展示Need Available部分)
3. 申请资源并生成安全序列(生成下方表格)：过程中需要手动输入第一个满足要求的Need的进程号(int)和各资源数量
